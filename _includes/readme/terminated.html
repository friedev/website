Slay robots in this top-down arena shooter inspired by Devil Daggers by Sorath and 2DDD by Bintr.

TERMINATED (including all code, art, and music) was created by Aaron Friesen.

Instructions

  • [R] to start
  • [ESC] to quit
  • [WASD] to move
  • Hold [LMB] to shoot
  • Tap [LMB] for shotgun
  • [RMB] for laser
  • [SCROLL] to zoom
  • [F] to toggle FPS
